# Dinogame-in-Docker-and-AWS-EC2

Description: Cloud deployment project of a Dino Game built with HTML/CSS/JavaScript, containerized using Docker and hosted on an Amazon EC2 instance from Amazon Web Services, demonstrating DevOps practices, containerization, and cloud infrastructure.

Tools used:
1 - Docker
2 - Html
3 - Linux OS
4 - AWS EC2

Context: Cloud engineers at a Big Tech company are facing excessive workloads, including activity flow monitoring, container creation and management, load balancing, and maintaining complex cloud architectures, while also needing productive downtime to improve efficiency and operational sustainability.
